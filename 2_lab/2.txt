Выполнить задания 1-2 с использованием оператора LIKE
1.	Вывести имена покупателей, которые начинаются и заканчиваются на букву v.
select name from cust where name ilike 'v%v';

2.	Вывести названия городов из таблицы продуктов, в которых вторая буква о, отсортированные в обратном порядке.
select city from prod where city like '_o%' order by city desc;

Выполнить задания 3-4 с использованием строковых функций
3.	Вывести в верхнем регистре все города из таблицы продавцов, имеющие длину более 6 символов. 
Пробелы в названиях городов должны быть удалены, а названия выведены по алфавиту.
select upper(replace(city, ' ','')) from sal where length(city) > 6 order by city asc;

4.	Вывести в одном столбце номера покупателей, соединенные с его именем и городом, 
в формате: ‘2001-IVANOV-PERM, только для тех покупателей, чье имя содержит подстроку ‘ov’.
select upper(concat(cnum, '-', name, '-', city)) from cust where name like '%ov%';

Выполнить задания 5-6 с использованием функций работы с датами
5.	Вывести временной интервал между сегодняшней датой и датой самого раннего заказа.
select age(current_date, ord_date) from ord order by ord_date asc limit 1;

6.	Вывести текущее время в формате 12тичасовых часов, 
с обозначением “am” для первой половины суток и “pm” – для второй.
select to_char(localtime, 'HH12:MI:SS AM') as time_12h; 

as time_12h - добавляет название

select name from my_schema.cust where name ~'^[A-I].*ov$';
